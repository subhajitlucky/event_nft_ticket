## ⚡ **Quick Start**

### 📋 **Requirements**
- Node.js `v18+`
- MetaMask Wallet
- Sepolia ETH for testing

### 🛠️ **Installation**
```bash
# Clone repository
git clone https://github.com/subhajitlucky/eventNftTicket.git
cd eventNftTicket

# Setup blockchain
cd blockchain && npm install

# Setup frontend  
cd ../frontend && npm install

# Start development
npm run dev
```
